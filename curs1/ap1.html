In exemple mai vechi de JavaScript declararea variabilelor se face cu var peste tot. Aceasta este varianta veche pentru declararea unei variabile, si nu este recomandata. Sunt cateva diferente subtile intre let si var

var VS let

  1. var: se poate redeclara aceasi variabila
  2. var: functie scop
  3. var: daca se acceseaza o variabila inainte de declarere : undefined
  4. var: toate browserele suporta
  1. let: NU se poate redeclara aceasi variabila
  2. let: bloc {} scop
  3. let: daca se acceseaza o variabila inainte de declarere : ReferenceError
  4. let: cateva browsere nu suporta

Let vs Const

  1. let si const sunt la fel
  2. singura diferenta este ca let poate sa isi schimbe valoarea
